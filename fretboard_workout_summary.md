## Core Organizing Concept: The Five Root Shapes (CAGED-equivalent)

Every scale, chord, arpeggio, and mode in the book is anchored to **5 root shapes** that repeat up the neck in the same order in every key. Each shape is defined by where roots fall on the strings:

| Pattern | Root strings |
|---------|-------------|
| 1 | 2nd & 5th, two frets apart |
| 2 | 5th & 3rd, two frets apart |
| 3 | 3rd, 1st, & 6th, three frets apart |
| 4 | 1st, 6th & 4th, two frets apart |
| 5 | 4th & 2nd, three frets apart |

Patterns overlap: pattern 1 shares a root on the 5th string with pattern 2, etc. The five shapes cycle continuously up the fretboard. This is the book's "Rosetta Stone" — everything else is derived from it.

**Trainer relevance:** This is exactly what the existing scale trainer is built on. The note-naming, interval, chord, and arpeggio games below all hinge on fluency with these shapes.

## Building Blocks (in pedagogical order)

### 1. Half Steps and the Musical Alphabet
- Only 12 notes; named using 7 letters (A–G) + accidentals (#/b/♮)
- **Natural half steps:** B→C and E→F (no black key between them on piano)
- Whole step = 2 frets; Half step = 1 fret
- Exception when crossing from 3rd string to 2nd string: whole step = 2 frets (not 3), half step = 3 frets (not 4) — because the guitar is tuned in 4ths except the G→B string (major 3rd)
- **Alphabet rule for scales:** all 7 letters used exactly once; no mixing sharps and flats

### 2. Note Naming
Two methods to name any note on the fretboard:
1. **Count from open string** using natural half steps (B→C, E→F)
2. **Root shapes** — if you know one note, nearby notes in the same root shape share names

Key anchor note to memorize beyond open strings: **D at the 7th fret of the 3rd string**

### 3. Major Scale Formula
`W W H W W W H` (degrees: 1 2 3^4 5 6 7^8 — carets mark half steps)

Five fingering patterns map directly onto the five root shapes. Each pattern has 3 notes per string except one string per pattern with 2 notes.

### 4. Natural Minor Scale Formula
`W H W W H W W` (degrees: 1 2^b3 4 5^b6 b7 8)

**Relative minor/major relationship — two key rules:**
- "The 6th degree of a major scale is the root of its relative minor."
- "The 3rd degree of a minor scale is the root of its relative major."
- The fingering patterns are identical; only the root note changes.

### 5. Pentatonic Scales
- **Major pentatonic:** major scale minus the 4th and 7th degrees → always 2 notes per string
- **Minor pentatonic:** natural minor minus the 2nd and 6th degrees → always 2 notes per string
- Relative pentatonic rule: same as major/minor — 6th degree of major pent = root of relative minor pent

### 6. Intervals
Intervals described as if the lower note were the root of a major scale.

**Perfect intervals** (1, 4, 5, 8): become augmented (+½ step) or diminished (−½ step)  
**Major intervals** (2, 3, 6, 7): become augmented (+½ step) or minor (−½ step) or diminished (−whole step)

Quality table:
```
Augmented
  ↕ (±½ step)
Major / Perfect
  ↕ (±½ step)
Minor / Diminished
```

Tritone (aug4 / dim5) = 3 whole steps — special symmetric interval.

**Guitar shapes for common intervals:**
- Unison: 5 frets down on adjacent string (4 frets on 3rd→2nd)
- M2: 3 frets down (2 frets on 3rd→2nd)
- M3: 1 fret down (same fret on 3rd→2nd)
- P4: same fret on next string (1 fret up on 3rd→2nd)
- P5: 2 frets up on adjacent string (3 frets on 3rd→2nd); 3-fret shape when skipping a string
- M6: 4 frets up on adjacent string (5 frets on 3rd→2nd); same fret when skipping
- M7: 1 fret up when skipping a string
- Octave = one of the 5 root shapes

### 7. Compound Intervals
Intervals larger than an octave; share quality with their simple-interval equivalent:
- 9th = 2nd + octave
- 10th = 3rd + octave
- 11th = 4th + octave
- 12th = 5th + octave
- 13th = 6th + octave

Used extensively in chord extensions (see below).

### 8. Triads
Four types:
| Type | Spelling | Symbol examples |
|------|----------|-----------------|
| Major | 1, 3, 5 | C, Cmaj, Cma |
| Minor | 1, b3, 5 | Cm, Cmin, C- |
| Diminished | 1, b3, b5 | Cdim, C° |
| Augmented | 1, 3, #5 | Caug, C+ |

**Inversions:**
- Root position: root is lowest note
- 1st inversion: 3rd is lowest
- 2nd inversion: 5th is lowest

**Close-voiced triads** (one note per string) form 3 repeating shapes across the full neck.  
**Open-voiced "cowboy" / barre chords** are triads with notes doubled in different octaves.

### 9. Seventh Chords
| Type | Spelling | Symbol |
|------|----------|--------|
| Major 7 | 1, 3, 5, 7 | Cma7, CA7 |
| Minor 7 | 1, b3, 5, b7 | Cmi7, C-7 |
| Dominant 7 | 1, 3, 5, b7 | C7 |
| Minor 7b5 (half-dim) | 1, b3, b5, b7 | Cmi7b5, Cø7 |
| Diminished 7 | 1, b3, b5, bb7 | Cdim7, C°7 |

**Stock voicings** (drop-2 open voicings, 4 notes across 4 strings):

| Pattern | Stock voicing |
|---------|--------------|
| 4 | 1-7-3-5 |
| 2 & 5 | 1-5-7-3 |
| 3 | 5-1-3-7 |
| 1 | 3-7-1-5 and 1-3-7-1 |

Dominant 7 is the "most important sonority" — implied by chord letter + "7" alone (no other symbol needed).

### 10. Extended Chords (9ths, 11ths, 13ths)
- All extensions up to and including the named one are included; lower ones may be omitted for playability
- Extensions default to major/perfect unless marked otherwise
- **The 3–11 clash:** P4 (= 11th) clashes with major 3rd → use #11 instead, or omit the 11th, or omit the 3rd on dominant chords
- Practice: keep extensions on top of voicings; dropping the 5th (then root) saves space

### 11. Altered Chords
Alterations = augment or diminish a chord tone or extension. Most common on dominant chords:
- Possible alterations: b5, #5, b9, #9, #11, b13
- "Alt" chord = dominant with altered 5ths and/or 9ths; no natural 5ths or 9ths allowed
- b5 and #5 are mutually exclusive (same with b9/#9)

### 12. Modes
All 7 modes share fingering patterns with the major scale — only the root changes:

| Degree | Mode | Character |
|--------|------|-----------|
| I | Ionian | = Major scale |
| II | Dorian | minor flavor, raised 6th |
| III | Phrygian | minor flavor, b2 |
| IV | Lydian | major flavor, #4 |
| V | Mixolydian | major flavor, b7 |
| VI | Aeolian | = Natural Minor |
| VII | Locrian | diminished flavor, b2 + b5 |

**"Find the Ionian" method** to locate any mode:
1. Identify mode number (e.g. Mixolydian = 5th)
2. Count down to degree 1 using major scale intervals to find the relative major (Ionian)
3. Play the major scale pattern but start and stop on the modal root

**Typical modal chord progressions given:**
- Ionian: C G F G
- Lydian: Cma7 D
- Dorian: Cmi7 F9 Cmi7 Dmi7
- Mixolydian: C7 Bbma7
- Phrygian: Cmi Db
- Aeolian: Cmi Bb Ab Bb

### 13. Other Common Scales
Spelled relative to major scale (1 2 3^4 5 6 7^8):

| Scale | Formula |
|-------|---------|
| Blues | 1 b3 4 b5 5 b7 |
| Country blues ("blues relative") | 1 2 b3^3 5 (major pent + b3) |
| Harmonic minor | 1 2^b3 4 5^b6 7^8 (raised 7th vs. natural minor) |
| Melodic minor | 1 2^b3 4 5 6 7^8 (raised 6th AND 7th vs. natural minor) |
| Diminished (whole-half) | 1 2^b3 4^b5 5 b6^b7 7^8 |
| Dominant diminished (half-whole) | 1^b3^3^b5^5^6^b7^8 |

All support 5 fingering patterns via root shapes.

### 14. Other Common Chords
| Type | Spelling | Notes |
|------|----------|-------|
| Sus4 | 1, 4, 5 | 3rd replaced by 4th |
| Sus2 | 1, 2, 5 | 3rd replaced by 2nd |
| Major 6 | 1, 3, 5, 6 | |
| Minor 6 | 1, b3, 5, 6 | (major 6th interval even in minor) |
| 6/9 | 1, 3, 5, 6, 9 | no 7th |
| Add9 | 1, 3, 5, 9 | no 7th — distinct from dominant 9 and ma9 |
| Slash chords | triad/bass note | specifies inversion or extended voicing |

## Teaching Philosophy (relevant to gamification)

- **Draw / Describe / Play** — mental image before fingers. The book's core method: visualize first, verbalize second, play third.
- **Start on root, end on root** — every scale/arpeggio practice should start and stop on the root so the ear anchors to it.
- **Overlapping patterns** — each pattern shares notes with the next; that overlap is the key to connecting the neck.
- **All 5 patterns in all 12 keys** — 60 instances of any shape. Mastery means fluency across all of them.
- **Slow + deliberate beats fast + sloppy** — 15-min sessions twice daily, no mistakes.
- **Saying things aloud** burns them in. The book repeatedly asks students to verbalize what they're playing (pattern number, note name, interval name, chord name).

## Summary of Testable Concepts (ranked rough priority for games)

1. **Note naming** — name any note at a given string/fret position (builds on root shapes + half steps)
2. **Interval recognition** — identify or build intervals by fretboard shape or note names
3. **Scale degree identification** — play a scale pattern and identify which degree each note is (1–7)
4. **Triad spelling** — given root and type, find notes; or given notes, name chord type + inversion
5. **Seventh chord recognition** — identify chord type by spelling or voicing shape
6. **Mode identification** — given a chord progression, identify the mode; or find the Ionian from a modal root
7. **Relative major/minor** — given a key, find its relative (applies to pentatonics too)
8. **Scale construction** — apply a formula (W/H pattern) to build any scale from any root on one string
9. **Arpeggio tracing** — connect the 3–4 notes of a chord type across all 5 patterns
10. **Extended/altered chords** — spell a chord given name + alteration symbols
